Security Vulnerabilities

If you discover a security vulnerability in NSE, do not open a public issue. Use GitHub's private vulnerability reporting feature on the repository's Security tab. Given that NSE handles cryptographic key material, we take security reports seriously and will respond promptly.

OpenClaw Packages

For issues with the OpenClaw Python ecosystem (nostrkey, nostrwalletconnect, nostrcalendar, nostrsocial), see the OpenClaw support page.
