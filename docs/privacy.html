NSE / Privacy

Privacy Policy

Last updated: March 13, 2026

The short version: NSE libraries run entirely on your device. We don't collect telemetry, require accounts, or phone home. Your keys never leave your hardware.

Who We Are

NSE (Nostr Secure Enclave) is maintained by Humanjava Enterprises Inc. This policy covers the NSE libraries (across npm, PyPI, Swift Package Manager, and Maven) and the nse.dev website.

The Libraries

NSE libraries are open-source, distributed under the MIT license. When you install and use them:

  • No telemetry. The libraries do not collect usage data, analytics, crash reports, or any other information.
  • No network calls. NSE operates entirely locally. Key generation, encryption, and signing happen on your device. The only network activity is what your application does with the signed output.
  • No accounts. No registration, API keys, or user accounts required.
  • Keys stay local. Private keys are encrypted by your device's hardware enclave and stored in platform-native secure storage (Keychain, KeyStore, etc.). We never have access.

Hardware Enclave Interaction

NSE uses your device's hardware security module (iOS Secure Enclave, Android StrongBox/TEE, or software fallback) to protect keys. It's worth being explicit:

  • The hardware key never leaves the device. The P-256 wrapping key is generated inside the secure enclave and is non-exportable by design.
  • Biometric data stays in hardware. When biometric unlock is used, the biometric check happens entirely within the secure enclave. NSE never sees or processes biometric data.
  • The encrypted key blob is stored locally. The AES-GCM encrypted secp256k1 key is stored in platform-native secure storage on your device.

OpenClaw Integration

NSE is the cryptographic foundation for the OpenClaw ecosystem. The OpenClaw Python packages (nostrkey, nostrwalletconnect, nostrcalendar, nostrsocial) follow the same privacy principles: local-first, no telemetry, no accounts. See the OpenClaw documentation for details on each package.

The Website

nse.dev is a static site hosted on GitHub Pages. We use:

  • No cookies.
  • No analytics.
  • GitHub Pages hosting. GitHub may collect standard server logs. See GitHub's data collection policy.

Children

NSE libraries and this website are developer tools not directed at children under 13.
